client.login(client, name, password, entry)[source]¶

send the login information to the server allowing the user to sign-in or sign-up

Parameters:
  • client (socket.socket) – connection object with the server
  • name (str) – username of the user trying to login
  • password (str) – password of the account trying to login
  • entry (int) – code for sign-in or sign-up
Returns:

error code or success code

error_code 1: successful sign-in error_code 2: incorrect password error_code 3: username not found error_code 4: already sign-in somewhere else error_code 5: username already exists and cannot be used to make a new account :rtype: int
